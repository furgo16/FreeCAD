Placement object
================

The ``FreeCAD.Placement`` class defines an orientation (rotation) and a
position (base) in 3D space. It is used when no scaling or other distortion
is needed.

It is an alias for the class :class:`FreeCAD.Base.Placement`.

Please see the documentation for :class:`~FreeCAD.Base.Placement` for the full API details.
