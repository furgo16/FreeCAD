Vector object
=============

The ``FreeCAD.Vector`` class represents a 3D vector with X, Y, and Z components.
It is fundamental for representing points, directions, and translations in FreeCAD.
It is an alias for the class :class:`FreeCAD.Base.Vector`.

Please see the documentation for :class:`~FreeCAD.Base.Vector` for the full API details.
