Axis object
===========

The ``FreeCAD.Axis`` class represents an axis in 3D space, defined by a point and a direction. 
It is an alias for the class :class:`FreeCAD.Base.Axis`.

Please see the documentation for :class:`~FreeCAD.Base.Axis` for the full API details.
