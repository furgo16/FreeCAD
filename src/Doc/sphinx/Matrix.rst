Matrix object
=============

The ``FreeCAD.Matrix`` class represents a 4x4 transformation matrix
for 3D operations, allowing for rotation, translation, and scaling.
It is an alias for the class :class:`FreeCAD.Base.Matrix`.

Please see the documentation for :class:`~FreeCAD.Base.Matrix` for the full API details.
